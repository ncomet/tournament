= tournament app

== prerequisites

* localstack installed and running
* `jdk8` or later

== to build and launch the app

[source,bash]
----
./gradlew build
./runApp.sh
----

== to access swagger API doc

http://localhost:8080/swagger

== to run Karate integration tests while app is running

[source,bash]
----
cd it
./gradlew test
----
